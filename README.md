# 🏋️‍♀️ 당신의 홈트 관리 어시스턴트, SSAFIT

“오늘도 운동했나요?”  
혼자 하는 홈트가 3일 이상 가기 힘든 이유는 **기록·관리·동기부여**가 제대로 되지 않기 때문입니다.  
**홈트 관리 어시스턴트 SSAFIT**은은 영상 기반 홈트에 **PT 코칭과 게이미피케이션 요소**를 더해,  
유저가 꾸준히 운동 루틴을 만들 수 있도록 돕는 서비스입니다.

[Check Out our Project Notion!](https://bedecked-child-87b.notion.site/SSAFIT-by-2ab4f7753cf280a982f6d6404e9cfa9a?pvs=143)


---

## 🚀 핵심 가치

1. **운동 기록 관리** – 회원/영상/운동 데이터를 체계적으로 관리  
2. **개인 맞춤형 PT 보조** – 과거 운동 기록과 영상 기반 피드백, 추천  
3. **꾸준함을 만드는 동기부여** – 듀오링고·게임식 시스템으로 재미와 지속성 확보  

---

## 📦 주요 기능

### 1. 회원 관리 (회원 CRUD)

- 회원 정보 등록/조회/수정/삭제
- 기본 프로필 정보(나이, 성별, 키, 몸무게 등)
- 운동 목표 설정 (체중 감량, 근력 강화, 체력 향상 등)
- 개인별 누적 기록(운동 시간, 소모 칼로리, 출석일 수 등) 조회

---

### 2. 영상 관리 (영상 CRUD)

- 운동 영상 등록/조회/수정/삭제
- 영상 메타데이터 관리
  - 난이도 (초급/중급/고급)
  - 운동 부위 (전신, 상체, 하체, 코어 등)
  - 소요 시간, 필요 장비
- 추천 알고리즘에서 활용 가능한 태그 시스템

---

### 3. 커뮤니티 (자유 게시판)

- 운동 인증글, 후기, 질문 작성
- 댓글/답글 기능
- 인기 게시글, 최신글 정렬
- 운동 루틴 공유 및 팁 공유

---

### 4. 소셜 기능

- **친구 추가**: 함께 운동하는 친구 리스트 관리
- **재생목록 공유**: 내가 만든 운동 영상 플레이리스트를 친구와 공유
- 친구의 누적 운동 기록/출석 현황(선택적 공개) 확인

---

### 5. 운동 일기 캘린더

- 날짜별 운동 수행 기록 표시 (캘린더 뷰)
- 하루 단위 요약:
  - 수행한 운동 목록
  - 운동 시간
  - 소모 칼로리
  - 개인 메모 & 피드백
- 주간/월간 리포트 제공 (총 운동 시간, 빈도, 평균 난이도 등)

## 🧑‍🏫 “홈트 + PT” 부가 기능

### 1. 홈트 영상에 PT 요소 추가

- 영상 내 운동 세트/횟수/휴식 시간 데이터화
- 세트 시작/종료, 휴식 시작/종료 타이머 제공
- “몇 세트 남았는지”, “휴식 얼마나 남았는지” 직관적으로 보여주는 UI

### 2. 과거 운동 기록 기반 AI 추천

- 최근 일주일/한 달 운동 패턴 분석
- 유저의:
  - 선호 난이도
  - 선호 시간대
  - 선호 부위
- 에 맞춰 다음 추천 영상/루틴 제안

### 3. 실시간/반실시간 코칭 요소

- 추천된 운동에 따라:
  - 올바른 자세 가이드(텍스트/영상/이미지 가이드)
  - 개수 세기, 시간 카운트
  - 세트 간 휴식 시간 관리
- (확장) 스마트워치/센서 연동 시 자세·심박 기반 피드백도 고려

### 4. 업로드 영상 기반 피드백

- 유저가 자신의 운동 영상을 업로드하면:
  - 수행한 동작의 난이도/안정성/완성도에 대한 점수화(초기 버전은 자기평가 + 코멘트 중심)
  - 강도/시간 대비 효율에 대한 간단 리포트 제공 (예: “오늘은 상체 위주, 난이도 중간”)

### 5. 운동 평가 및 다음 추천에 반영

- 운동 종료 후 간단 설문:
  - 난이도 체감 (너무 쉬웠음 ~ 너무 힘들었음)
  - 오늘 컨디션
  - 운동 만족도
- 이 데이터를 다음 추천 영상/루틴에 반영 → 점점 나에게 맞는 PT처럼 진화

## 🎮 동기부여 & 게이미피케이션 요소

### 1. 연속 운동 일자 (Streak)

- **듀오링고식 연속 출석 시스템**
- 연속 운동 일수에 따라:
  - 뱃지/칭호 지급
  - 특별 미션 오픈
- Streak 끊김 방지를 위한 “휴식권” 기능도 추후 고려

### 2. 누적 운동 시간 & 칼로리

- **게임의 누적 플레이 시간**처럼:
  - 지금까지 내가 운동에 쓴 총 시간
  - 누적 소모 칼로리
- 마일스톤마다:
  - 예: 10시간 달성, 50시간 달성, 100시간 달성 뱃지 지급

### 3. 소규모 운동 커뮤니티 / 인증 모임

- 길드 같은 **소규모 운동 그룹**
- 그룹별:
  - 주간 공동 목표 (예: “이번 주 전체 500분 운동 달성!”)
  - 그룹 인증 게시판
- 서로의 운동 기록을 보며 응원/경쟁

### 4. 하루하루 미션 (Daily Quest)

- 예시:
  - 오늘의 미션: “전신 20분 이상 운동”, “스트레칭 포함”, “하체 루틴 1개 수행”
- 미션 완료 시:
  - 경험치/재화/뱃지 지급

### 5. 재화 기반 캐릭터·공간 꾸미기

- 운동 완료 → **재화 획득**
- 재화 사용처:
  - 나만의 아바타/캐릭터 꾸미기
  - 방/운동 공간 인테리어
- “운동할수록 내 캐릭터/공간이 성장하는” 재미 제공

## 👥 타겟 유저

- 혼자 홈트를 하다가 **금방 포기해본 경험이 있는 사람**
- 운동 기록과 데이터 보는 걸 좋아하는 **자기관리형 유저**
- 게임식 보상과 캐릭터 꾸미기를 좋아하는 **MZ 세대**
- 헬스장 대신 집에서 꾸준히 운동하고 싶은 직장인/학생

## 🧱 (예정) 기술 스택

> 실제 구현 단계에 따라 변경될 수 있습니다.

- **Frontend**: Vue
- **Backend**: Spring Boot
- **DB**: MySQL
- **AI 추천**: Python 기반 추천 모델 + 간단한 Rule-based 추천부터 시작
- **인증/보안**: JWT 기반 로그인, OAuth2 (구글/카카오 연동 등)
